# Designer Portfolio

A minimal designer portfolio website built with Astro framework.

## Tech Stack

- [Astro](https://astro.build/) - Modern static site framework
- [Tailwind CSS](https://tailwindcss.com/) - Utility-first CSS framework
- TypeScript - Type-safe JavaScript

## Getting Started

### Install Dependencies

```bash
npm install
```

### Development Server

```bash
npm run dev
```

Visit [http://localhost:4321](http://localhost:4321) to view the website.

### Build

```bash
npm run build
```

### Preview Build

```bash
npm run preview
```

## Project Structure
